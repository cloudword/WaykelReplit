Waykel is a production-ready ride-sharing platform (Uber/Ola equivalent) with:

2 Mobile Apps (Rider + Driver) built with React Native + Expo

3 Web Admin Dashboards (Transporter, Customer, Super Admin) built with React + Vite

1 Robust Backend (Node.js + Express) with real-time features

Complete Payment Integration (Stripe + Razorpay)

Real-Time Tracking (Socket.IO + Geospatial Queries)

Full Authentication System (JWT + Role-based Access)
